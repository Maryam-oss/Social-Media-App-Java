Please download the following MongoDB Java Driver JARs (version 4.x or 5.x) and place them in this folder:

1. mongodb-driver-sync.jar
2. mongodb-driver-core.jar
3. bson.jar
4. slf4j-api.jar (optional but recommended for logging)
5. slf4j-simple.jar (optional)

You can download them from Maven Central or the MongoDB website.
