# Social Media Application (Java)

A fully functional social media application developed as a final project for Object-Oriented Programming (OOP).

## Features
- Interactive feed (posts, likes, comments)
- One-to-one and group chat system
- User profiles and customization
- Smart notifications system
- Secure backend using DAO pattern

## Tech Stack
- Java (OOP, Collections, Multithreading)
- Java Swing (Modern UI)
- JDBC & Database
- MVC & DAO Design Patterns

## How to Run
1. Open the project in any Java IDE
2. Configure database connection
3. Run `Main.java`
